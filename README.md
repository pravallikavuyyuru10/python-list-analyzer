# python-list-analyzer
Beginner Python project that analyzes a list and finds the largest, smallest, and even-count values.
Features:
- Finds the largest number
- Finds the smallest number
- Counts even numbers

Concepts used:
- Lists
- Loops
- Conditions
- Variables

Sample Input:
[10, 25, 8, 99, 14]

Sample Output:
Largest: 99
Smallest: 8
Even Count: 3
